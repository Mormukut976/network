Program overview ▶

The CyberTeam Network Engineer program validates your ability to install, configure, operate, and troubleshoot medium-size routed and switched networks. This program covers a broad range of fundamentals, including:

  • Network fundamentals and access
  • Cisco CCNA
  • CCNA Security
  • Web Security
  • Web Security Essentials
  • Install & Configure
  • AR Security
